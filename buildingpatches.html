Patch building guide (PRP)

In this step, we will show you how to make your own simple compatiblity patches for PRP and common mods like Ultra Interior Lighting and Fogout.

This document assumes you have completed the guide up to at least the Bug Fixes page or later.

Start by downloading the files from PJM's Precombine Scripts mod page. Though the other scripts are useful in building the larger patches against PRP, for this purpose, you only need FO4Check_Previsibines.pas which the latest can be found under update files (4.3b) or optional files (4.4c) as of this writing. Copy the .pas file to your xEdit's installation, specifically the Edit Scripts subfolder.

Install the desired mod or mods to patch against through mod organizer 2 and enable the mod on the left side pane.

Locate the plugin associated with the mod that was just installed, dragging it before PRP.esp if required.

Start xEdit through mod organizer 2, as per usual. Wait for it to handle the usual cache and plugin loading tasks it does on startup.

Right click on PRP.esp, Apply Script, type fo4check in the filter box, then select FO4Check_PreVisbines in the list. Click OK if it's in the list of scripts, if not, check to see if the script is installed in the right place for xEdit to find it. Click the radio button with the text regarding Creating a Patch to only fix wrong Regions/MDHT/etc and also check Process complete Load Order. Hit okay, and let it cook. It'll take a bit depending on the depth of your load order and hardware. Onc it's complete and you see "Done: Applying script" in the Messages tab of xEdit, exit xEdit, it'll show a popup before exiting with a list of changed plugins, you want to hit OK here to save FinalCellPatch.esp.

At this point you can tell at a glance if it worked or not, as the resulting plugin, FinalCellPatch.esp, won't have any conflicts. If there's still unexpected issues like wrong lighting after installing a new mod, consider regenerating the plugin after deleting it as above. FinalCellPatch (or if you manually rename the plugin later, whatever the name is now) has to be below PRP.esp to have any effect. So, start the game and have fun!
